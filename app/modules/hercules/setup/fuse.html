Registering your Expressway Clusters with Fusion to enable Cloud Fusion services:

  1. In a new browser window, go to the web interface of the Expressway Cluster you want to fuse.
  2. Ensure all Expressways in the cluster are running the latest software version.
  3. On the Expressway Cluster, enable and configure the Cloud Fusion Services you want to use.
  4. Return to this page to confirm registration and continue with the Setup Wizard.

Each Expressway Cluster will appear in the list below once it is registered for Cloud Fusion, displaying the status of each configured service.
