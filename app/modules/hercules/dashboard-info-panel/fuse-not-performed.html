To-do Register Cisco VCS for Cloud Extensions

Register each Cisco VCS (or cluster) with Cisco Collaboration Cloud as follows. For clusters, you only need to register the master peer.

  1. Use a new browser window or tab to log on to the Cisco VCS (master peer if clustered).
  2. Go to Applications > Cloud Extensions > Connector Management and click Register. You should now see links to your Connectors from that page.

Visit the Cloud Extensions help site for more information.
