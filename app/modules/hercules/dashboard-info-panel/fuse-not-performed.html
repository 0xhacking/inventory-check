Register each Expressway (or Expressway cluster) with Cisco Collaboration Cloud as follows. For clusters, you only need to register the master peer.

  1. Keep this page open in your browser to monitor status throughout the process
  2. Open a new browser window or tab to sign in to the Expressway (master peer if clustered)
  3. Go to Applications > Cloud Extensions > Cloud Extensions Connectors and read the information.
  4. Modify the Expressway trust list if required. See deployment documentation.
  5. Set up an HTTP proxy if required. See deployment documentation.
  6. Click Register. One or more Connectors will download and install, according to your Service entitlements.
  7. Configure each connector, using the newly installed menu items on the Applications > Cloud Extensions menu
  8. Return to the Spark Management browser tab/window

See documentation for more details.
